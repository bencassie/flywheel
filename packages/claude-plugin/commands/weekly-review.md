# /weekly-review - Comprehensive Weekly Reflection

Generate a comprehensive weekly reflection from your vault activity.

## Usage

```
/weekly-review                 # This week
/weekly-review 2025-W01        # Specific week
```

## What It Does

```
Weekly Review Process
────────────────────────────────────────────────────────────────
Analyzing: 7 daily notes, 45 modified notes, 127 edits
────────────────────────────────────────────────────────────────
```

## Where Output Goes

| Action | Target | Notes |
|--------|--------|-------|
| Review | Console + weekly note | Full reflection |
| Summary | Weekly note file | Saved summary |

